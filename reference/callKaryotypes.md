# Take prior and observed counts and calls karyotypes and Z-scores

Take prior and observed counts and calls karyotypes and Z-scores

## Usage

``` r
callKaryotypes(indat, inDirichlet, p_contamination = 0.1)
```

## Arguments

- indat:

  full input tibble

- inDirichlet:

  associated Dirichlet parameters

- p_contamination:

  the assumed probability of contamination

## Value

karyotype calls
