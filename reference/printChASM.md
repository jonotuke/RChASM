# A function to print the result of the combined analysis to screen

A function to print the result of the combined analysis to screen

## Usage

``` r
printChASM(inChASM, lines = 20)
```

## Arguments

- inChASM:

  the result of a full ChASM analysis (output from runChASM)

- lines:

  the number of lines to print to screen

## Value

print output
