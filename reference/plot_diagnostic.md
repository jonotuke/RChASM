# Plots diagnostic plots

Plots diagnostic plots

## Usage

``` r
plot_diagnostic(IDs, inChASM, addLabels = FALSE)
```

## Arguments

- IDs:

  a list or predefined piped string of sample IDs to look for

- inChASM:

  parameter object

- addLabels:

  add "A", "B" and "C" to the plot panels?

## Value

plots
